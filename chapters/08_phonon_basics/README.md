# 第八章：声子计算与晶格动力学

## 1. 声子物理概念

### 1.1 晶格振动与简正模式

晶体中的原子并非静止不动，而是围绕平衡位置做微小振动。当我们将原子位移展开为**简正模式（normal modes）** 时，每个模式以特定频率 $\omega$ 独立振荡。对于含 $N$ 个原子的原胞，共有 $3N$ 个简正模式（其中 3 个为声学支，对应整体平移）。

> **关键概念：声子**
>
> **声子（phonon）** 是晶格振动的量子化描述。每个简正模式对应一种准粒子——声子，其能量为 $\hbar\omega$。声子是玻色子，遵循玻色-爱因斯坦统计，是理解固体热容、热导率、超导等性质的核心。

### 1.2 声子色散关系

声子频率随波矢 $\mathbf{q}$ 的变化关系 $\omega(\mathbf{q})$ 称为**声子色散关系**。色散关系包含了晶格动力学的全部信息：

- **声学支**（acoustic）：$\omega \to 0$ 当 $\mathbf{q} \to 0$，描述原子同向运动
- **光学支**（optical）：高频段，描述原子反向运动，在红外光谱中有响应
- **LO-TO 分裂**：极性晶体中纵光学支（LO）与横光学支（TO）在 $\Gamma$ 点处的频率劈裂

## 2. 密度泛函微扰理论（DFPT）原理

Quantum ESPRESSO 中的声子计算基于**密度泛函微扰理论（Density Functional Perturbation Theory, DFPT）**。其核心思想是：

1. 晶体处于平衡构型时，每个原子受力为零
2. 对原子位置施加无穷小位移（微扰），计算电子密度的线性响应
3. 由响应函数构建**动力学矩阵（dynamical matrix）**
4. 对角化动力学矩阵得到声子频率

DFPT 的优势在于：
- **无需超胞**：直接在原胞中计算任意 $\mathbf{q}$ 点的声子
- **系统性收敛**：精度完全由平面波截断能和 $\mathbf{k}$ 点网格决定
- **可同时计算响应性质**：介电常数、Born 有效电荷等

## 3. 完整声子色散计算流程

声子色散计算分为四个步骤：

```
┌─────────────┐    ┌─────────────────┐    ┌──────────────┐    ┌───────────────┐
│  Step 1:    │    │  Step 2:        │    │  Step 3:     │    │  Step 4:      │
│  pw.x SCF   │───>│  ph.x           │───>│  q2r.x       │───>│  matdyn.x     │
│  自洽计算    │    │  多 q 点声子计算 │    │  傅里叶逆变换 │    │  色散曲线计算  │
└─────────────┘    └─────────────────┘    └──────────────┘    └───────────────┘
     电荷密度         D(q) 动力学矩阵       C(R) 实空间力常数      ω(q) 色散关系
```

### 3.1 Step 1：SCF 自洽计算（pw.x）

首先进行标准的 SCF 计算，获得基态电荷密度。这一步的要点：

- `prefix` 和 `outdir` 将在后续步骤中复用
- K 点网格需足够密以保证电荷密度收敛
- 保存波函数和电荷密度供 ph.x 读取

### 3.2 Step 2：ph.x 声子计算

ph.x 在 SCF 电荷密度的基础上，计算指定 $\mathbf{q}$ 点处的动力学矩阵 $D(\mathbf{q})$。

**关键参数**：

| 参数 | 含义 | 说明 |
|------|------|------|
| `tr2_ph` | 自洽收敛阈值 | 默认 1.0d-12；声子计算通常比 SCF 更严格，推荐 1.0d-14 |
| `prefix` | 计算前缀 | **必须与 SCF 计算完全一致** |
| `outdir` | 输出目录 | **必须与 SCF 计算完全一致** |
| `amass(i)` | 原子质量 (amu) | 可选，不指定则从赝势文件读取 |
| `fildyn` | 动力学矩阵输出文件名 | 默认 `'matdyn'`；建议命名含 q 点信息 |
| `epsil` | 是否计算介电常数 | 仅在 $\mathbf{q}=0$ 且为非金属时设为 `.true.` |
| `trans` | 是否计算声子 | 默认 `.true.`；设为 `.false.` 则只计算介电性质 |
| `ldisp` | 是否计算 q 点网格 | 设为 `.true.` 时配合 `nq1, nq2, nq3` 使用 |

**q 点输入格式**：在 `&INPUTPH` namelist 之后，单独一行给出 $\mathbf{q}$ 点坐标（笛卡尔坐标，单位 $2\pi/a$）：

```
0.0  0.0  0.0    # Gamma 点
1.0  0.0  0.0    # X 点
```

### 3.3 Step 3：q2r.x 傅里叶逆变换

将 $\mathbf{q}$ 空间的动力学矩阵 $D(\mathbf{q})$ 通过傅里叶逆变换得到实空间力常数 $C(\mathbf{R})$。

| 参数 | 含义 |
|------|------|
| `fildyn` | ph.x 输出的动力学矩阵文件名（含 `.0` 后缀的主文件） |
| `flfrc` | 输出的实空间力常数文件 |
| `zasr` | 声学求和规则类型：`'no'`（不施加）、`'simple'`、`'crystal'`、`'zero-dim'` |

> **关键概念：声学求和规则（ASR）**
>
> 声学求和规则要求零频声学模式在 $\mathbf{q}=0$ 时频率严格为零。数值误差会导致非零的"虚频"声学支，施加 ASR 可以修正这类误差。常用 `'crystal'` 或 `'simple'`。

### 3.4 Step 4：matdyn.x 计算色散曲线

利用实空间力常数，插值得到任意 $\mathbf{q}$ 点的声子频率，从而绘制完整的色散曲线。

| 参数 | 含义 |
|------|------|
| `flfrc` | q2r.x 输出的力常数文件 |
| `q_in_band_form` | `.true.` 时按高对称路径格式输入 q 点 |
| `asr` | 声学求和规则，与 q2r.x 中一致 |
| `flfrq` | 频率输出文件（默认 `'matdyn.freq'`） |
| `flvec` | 振动模式输出文件（默认 `'matdyn.modes'`） |

## 4. Gamma 点声子的特殊性

$\Gamma$ 点（$\mathbf{q}=0$）是唯一可以在 SCF 计算后**直接运行** ph.x 的 $\mathbf{q}$ 点。原因在于 $\Gamma$ 点的波函数与 SCF 使用的 $\mathbf{k}$ 点网格完全重合，无需额外准备。

对于其他 $\mathbf{q}$ 点（如 X、L 等），ph.x 需要读取 SCF 的电荷密度后自行进行非自洽计算，因此计算量更大，但**无需手动运行 nscf 计算**——ph.x 会自动处理。

> **注意**：ph.x 运行后会覆盖 SCF 的波函数文件。如果后续还需要用到 SCF 的波函数，应在 ph.x 运行前备份 `outdir` 中的数据。

---

## 5. 介电性质计算：epsil 与 Born 有效电荷

在 $\Gamma$ 点声子计算中设置 `epsil=.true.` 可以额外计算：

- **电子介电常数张量** $\varepsilon_\infty$：描述晶体对电场的电子响应
- **Born 有效电荷** $Z^*$：描述原子位移引起的宏观极化变化

这些量对于计算 LO-TO 分裂（极性光学声子在 $\Gamma$ 点的劈裂）至关重要。计算条件：

- **非金属体系**（绝缘体或半导体）
- **仅限 $\mathbf{q}=0$**
- `trans=.true.`（默认）且 `epsil=.true.` 时自动计算 Born 有效电荷

---

## 6. 动力学矩阵文件格式

ph.x 输出的动力学矩阵文件（如 `si.dynG`）包含：

- 晶格信息（原胞矢量、原子位置、原子类型）
- $\mathbf{q}$ 点坐标
- 动力学矩阵 $D_{\kappa\alpha,\kappa'\beta}(\mathbf{q})$ 的实部和虚部
- 若计算了介电性质，还包含 $\varepsilon_\infty$ 和 $Z^*$

文件结构示例（简化）：

```
Dynamical matrix file
... 晶格矢量 ...
... 原子坐标 ...
q = (  0.0000000  0.0000000  0.0000000 )
   1    1   real part of dyn. matrix
   ... 3x3 块 ...
   1    1   imag part of dyn. matrix
   ...
```

每个 $\mathbf{q}$ 点的矩阵维度为 $3N \times 3N$（$N$ 为原胞原子数）。

## 7. 声子计算的并行策略

声子计算中最高效的并行方式是 **q 点并行**：

- 使用 `ph.x` 的 `-npool` 选项将不同的 $\mathbf{q}$ 点分配到不同进程组
- 例如：`mpirun -np 16 ph.x -npool 4 < input` 将 16 个进程分为 4 组，每组处理不同的 q 点
- 使用 `ldisp=.true.` 时，所有不等价 q 点会自动分配

其他并行层次：
- **K 点并行**（`-nk`）：在每个 q 点内部，k 点可以并行
- **带并行**（`-nb`）：不同能带的并行计算

> **性能提示**：对于声子色散计算，q 点数量通常远大于 k 点数量，因此 `-npool` 是最有效的并行策略。

---

## 8. tr2_ph 收敛阈值

`tr2_ph` 控制 ph.x 内部自洽循环的收敛精度，类似于 pw.x 中的 `conv_thr`。

| tr2_ph | 精度等级 | 适用场景 |
|--------|----------|----------|
| 1.0d-10 | 粗略 | 快速测试、初步探索 |
| 1.0d-12 | 标准 | 常规声子计算（默认值） |
| 1.0d-14 | 高精度 | 本教程推荐值；介电性质计算 |
| 1.0d-16 | 极高精度 | 需要极高数值精度的特殊场景 |

> **注意**：过松的 `tr2_ph` 会导致声子频率不准确，尤其是低频声学支可能偏离零频较大。
